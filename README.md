# Scientific-Calculator
Scientific Calculator made using Python 
A Python-based scientific calculator built as a beginner project.

## Features
- Basic operations: +, -, *, /
- Power and modulus (%)
- Square root, logarithm, absolute value
- Trigonometry: sin, cos, tan (as degrees and not as radians)
- Handles divide by zero errors
- Returns invalid for other operations excluding these

## How to Run
1. Clone the repository
2. Run `python scientific_calculator.py` in your terminal
   OR open in Google Colab directly

## Tech Used
- Python 3
- `math` module
